How to Use
  1. Enter the Exam Name.
  2. Select a Version Number.
  3. Choose Yes or No to randomise the order of questions within each section.
  4. Drag and drop Question Files into the application. A section will be created in the exam script for every file provided. (only .txt files are allowed).
  5. Select the Output folder.
  6. Go to Exam > Generate

Question File Format

 
Line 1: Section Title*
Line 2: Number of Questions
[
  Line 3: Question Data (Question Number : Number of Options : Question)
  [
    Line 4: Correct answer first
    Line 5: Incorrect option
    Line 6: More incorrect options
    ... (determined by [Number of Options] set above
  ]
]

Rules: follow the rules to ensure the generator does not run into any problem

  • "Section Title" can also be HTML code (must start with <h1>)
  • Leave a blank line for title if no title is required
  • The number of questions will determine how many questions will be read from the file
  • Square brackets [] means "one or more of..."
  • Correct answer must be placed first, even if it is "None of the above" or "All of the above"
  • If the option "None/All of the above" is incorrect, place it last
  • Line 3-6 may contain converters (see below) for special format

Special Format Converters
#[Code here]#
code in a stand alone paragraph
#{Inline code here}#
inline code
#<image filename>#
place an image (stand alone paragraph)
#T
insert tab (within code)
#B
new line in multi-line code
#N
insert new line and indent
