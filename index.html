    Exam Generator [2015]

    When I was working as the teaching assistant in the IT department in Invercargill, NZ, I saw the opportunity to automate the process of producing multi-choice exam papers. There wasn't many available solution available to suited my need so I created one that was simple to use and easy to format. The final solution was a Java based application which took in a textfile for input, and outputted HTML files (Questions and Answers) which can be easily printed. If you are sick of randomising test questions and want it done automatically with only a few clicks, give Exam Generator a try!
